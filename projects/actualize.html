Brief Summary

Actualize's Customer Relationship Management (CRM) system was a fullstack web application that allowed their administrators to follow up with leads (i.e. people interested in joining the bootcamp), create leads of their own, and track their progress.

Technologies

Twilio

Cloud communications platform for building SMS, Voice & Messaging applications.

Vue.js

A JavaScript library for building user interfaces.

Ruby on Rails

A server-side web application framework written in Ruby.

Overview



This project was part of the Actualize apprenticeship program where I was able to work with a software development team in an agile-environment, adding features to the real CRM that Actualize uses to manage its prospective applicants.

My contributions to the project consisted primarily of integrating the Twilio API functionality and assisting in restructuring the frontend to be a single-page application. During my time on the project, I would consistently work with other programmers completing tickets geared towards these accomplishments.

We integrated the Twilio API as a way to improve our administrator's methods of reaching candidates, since it would allow them to send text messages to their leads directly through the application. We registered a TWILIO_ACCOUNT_SID, and TWILIO_AUTH_TOKEN, set them as environment variables for security, routed the correct actions to the leads_controller and lastly created the proper frontend forms.

The reason why we decided to take the time to restructure the frontend as a single-page application was to minimize page reloads and create a more fluid user experience. We used the Vue.js framework for this restructuring where I helped construct the nav bar component and integrate the bootstrap framework.
