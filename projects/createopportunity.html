Brief Summary

Create Opportunity is a nonprofit aimed at providing micro-loans to local Bay Area entrepreneurs and business owners with zero interest rates. This website was a static site to provide information on the venture and provide viewers with a way to contact us.

This venture was spearheaded by Daniel Engels, who was responsible for reaching out to investors and potential applicants, while I, on the other hand, was in charge with the communication of the brand.

Technologies

HTML5

An open-source JavaScript framework for building user interfaces.

PHP

A server-side web application framework written in Ruby.

Overview




With speed in mind, I was able to pull a template for the general layout of the website, make the appropriate alterations to how the components were laid out, build the appropriate pages and connect the site to a Heroku server, where the PHP code embedded into the template was able to be executed.

In terms of time, the task that took the longest was designing the overall site and filling it in with the appropriate content.

Where is the project now? It is still in development. We are still in the process of figuring out the most cost-efficient way to recieve donations through the site before connecting it to our purchased domain name.
